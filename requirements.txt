# Ce projet est une application Node.js (React/TypeScript).
# Il n'utilise pas Python, les dépendances ne s'installent donc pas avec `pip install -r requirements.txt`.

# Pour installer les dépendances et utiliser ce projet, vous devez utiliser npm (Node Package Manager).
# Veuillez suivre ces instructions :

# 1. Installez Node.js (si ce n'est pas déjà fait)
# 2. Ouvrez un terminal dans ce dossier
# 3. Exécutez la commande suivante pour installer toutes les dépendances listées dans package.json :
npm install

# 4. Pour lancer l'application en mode développement, exécutez :
npm run dev

# (Principales dépendances utilisées : React, Vite, Supabase, Tailwind CSS, Radix UI)
